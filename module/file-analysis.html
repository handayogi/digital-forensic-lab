File Analysis

Apa itu File Analysis?

Introduction

File analysis atau analisis file merupakan tahap awal yang krusial dalam Digital Forensic. Analisis file merupakan proses pemeriksaan file digital untuk mengidentifikasi struktur, konten, dan artefak yang relevan dalam investigasi forensik digital. Analisis ini tidak hanya berfokus pada isi file, tetapi juga pada karakteristik teknis seperti format, signature, dan atribut file.

Artefak dalam Digital Forensic adalah sisa jejak, data, atau informasi digital yang tertinggal dari aktivitas pengguna pada sistem atau perangkat digital, yang dapat digunakan untuk:

  • Mengetahui apa yang sebenarnya terjadi
  • Mengetahui kapan kejadian terjadi
  • Mengetahui bagaimana suatu aktivitas dilakukan

Melalui file analysis, investigator dapat menganalisis karakteristik teknis seperti format, signature, dan atribut file untuk memastikan keaslian dan integritas data digital.

Lesson Overview

  • File memiliki struktur internal (header, body, footer)
  • Ekstensi file tidak selalu mencerminkan format sebenarnya
  • File dapat dimodifikasi atau dimanipulasi
  • File analysis menjadi dasar analisis forensik lanjutan

Learning Outcomes

  • Memahami konsep dasar analisis file digital
  • Mengidentifikasi struktur dan karakteristik file
  • Menentukan peran file sebagai artefak forensik

Practice Scenario

Langkah 1 - Pemilihan Bukti

Periksa bukti digital dalam directory evidence. Pastikan jumlah, nama, dan size file yang tidak biasa.

cd evidence/file_analysis
ls -lah

Fungsi dari perintah atau command ls -lah:

  • l, menampilkan detail file (permission, owner, size, timestamp).
  • a, menampilkan semua file, termasuk file yang tersembunyi.
  • h, menampilkan file dalam format human-readable.

Langkah 2 - Verifikasi Tipe Asli File

Identifikasi apakah ekstensi file sama dengan tipe asli nya.

file *

Fungsi dari perintah atau command file *:

  • Mengidentifikasi tipe file sebenarnya berdasarkan signature (magic bytes), bukan ekstensi.

Langkah 3 - Analisis Metadata & Timestamp

Analisis metadata file untuk mengindentifikasi keanehan pada timeline.

stat {nama_file}
exiftool {nama_file}

Fungsi dari perintah atau command exiftool {nama_file}:

  • Mengidentifikasi metadata digital seperti waktu, lokasi, perangkat, dan aktivitas terkait suatu objek digital (khusus untuk file media & dokumen).

Langkah 4 - File Header Inspection

Konfirmasi keaslian dari file bukti digital dengan melakukan inspeksi pada header.

xxd {nama_file} | head -n 20

Fungsi dari perintah atau command xxd {nama_file} | head -n 20:

  • xxd -> hexdump, menampilkan representasi heksadesimal dari file digital.
  • head -n 20, menampilkan 20 baris pertama dari file digital.

Langkah 5 - Temukan Data Tersembunyi

Identifikasi atau deteksi file tersembunyi yang terkandung dalam file.

binwalk {nama_file}
binwalk -e {nama_file}

Fungsi dari perintah atau command binwalk {nama_file}:

  • Mendeteksi file atau data tersembunyi di dalam file lain.
  • -e, mengestrak otomatis file tersembunyi yang terdeteksi.
